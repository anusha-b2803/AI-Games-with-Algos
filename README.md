# AI Game Playground 🎮🤖

An interactive, browser-based repository of classical **Artificial Intelligence Algorithms** modeled through hands-on games. Built with pure Vanilla JS, the playground helps learners visualize how state spaces are searched, backtracked, and optimized in real time.

## ⚡ Key Features

*   **100% Client-Side**: No servers, no APIs, no transpiles. Runs absolute native and instant anywhere.
*   **Live Activity Logs**: Sidebar trace Streams demonstrating step-by-step state visits and constraints checks during solving.
*   **Zero Dependencies**: Written using absolute Vanilla HTML5, CSS3, and JavaScript primitives.
*   **Unified Dashboard View**: Tabbed layout for buttery smooth navigations switches cleanup workflows.

## 🧠 Solvers & Games

### 1. Tic-Tac-Toe AI (**Minimax**)
*   Uses recursive **Minimax with Alpha-Beta Pruning** sweeps.
*   Evaluate absolute winning states branches safely.
*   Two modes available: vs Player or unbeatable AI.

### 2. Sudoku Solver (**Backtracking + MRV**)
*   Uses heavy Constraint Satisfaction propagation.
*   Optimized with **MRV (Minimum Remaining Values)** heuristics solving sparse, hard boards almost instantly.
*   Traces tried numbers frame by frame visually.

### 3. Maze Pathfinder (**A\* Search**)
*   Grid Canvas driven graph searches.
*   Uses **Manhattan Distance Heuristics** resolving optimal shortest pathway animations.
*   Supports live draw, erase or random setup generations.

### 4. N-Queens Solver (**Constraint checking Backtracking**)
*   Animated visualizer resolving N row configurations properly.
*   **Randomized Branch sweeps**: Solves unique valid placements layout configurations with each run visually.
*   Supports cycle navigation on multiple discovered solution trees.
