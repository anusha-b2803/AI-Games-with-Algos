AI Game Playground

Interactive Artificial Intelligence Algorithm Playground

This project demonstrates four classical AI algorithms through interactive browser-based games. Each game lets you watch the algorithm think in real time — exploring states, backtracking, and converging on solutions with full visualization.

The entire project runs client-side with zero dependencies — no libraries, no backend, no build tools. Pure HTML5, CSS3, and Vanilla JavaScript.
