🧠 Minimax Algorithm
▾

Minimax is a decision-making algorithm used in two-player games. It assumes both players play optimally — the AI maximizes its own score while the opponent minimizes it.

The algorithm builds a game tree exploring all possible moves to a terminal state (win/draw/lose), then picks the move leading to the best outcome.

✂️ Alpha-Beta Pruning
▾

Alpha-Beta pruning cuts branches in the game tree that cannot affect the final decision. If the current branch is already worse than a known option, skip it.

α = best for maximizer β = best for minimizer if β ≤ α → prune branch ✂️

In the best case, pruning reduces nodes from O(bd) to O(bd/2).

📋 Algorithm Steps
▾
  1. Generate all possible moves from current state
  2. For each move, recursively evaluate the game tree
  3. Alternate between maximizing (AI) and minimizing (opponent)
  4. Apply alpha-beta cutoffs to prune dead branches
  5. Return the move with the highest minimax score
⏱ Complexity
▾
Minimax Time O(b^d)
With Alpha-Beta O(b^(d/2))
Space O(d)
Branching factor (b) ≤ 9
Max depth (d) 9
