♛ N-Queens Problem
▾

Place N chess queens on an N×N board so that no two queens attack each other — they cannot share the same row, column, or diagonal.

The 8-Queens puzzle has 92 solutions. For N=1: trivially 1. For N=2,3: no solutions!

↩️ Backtracking Approach
▾

We place one queen per row, scanning columns left to right:

isSafe(board, row, col): check same column ↑ check left diagonal ↗ check right diagonal ↖ all clear → safe to place
📋 Algorithm Steps
▾
  1. Start at row 0, try each column
  2. Check if placement is safe (no attacks)
  3. If safe: place queen, advance to next row
  4. If conflict: highlight and skip this column
  5. If no column works in this row: backtrack to previous row
  6. Remove queen from previous row, try next column
  7. Continue until all N queens placed (solution!)
