About This Project

An interactive AI algorithm lab built entirely in vanilla HTML/CSS/JS

⬡

AI Game Playground

Interactive Artificial Intelligence Algorithm Lab

This project demonstrates four classical AI algorithms through interactive browser-based games. Each game lets you watch the algorithm think in real time — exploring states, backtracking, and converging on solutions with full visualization.

The entire project runs client-side with zero dependencies — no libraries, no backend, no build tools. Pure HTML5, CSS3, and Vanilla JavaScript.

Games

What's Included

✕○ Minimax

Tic Tac Toe AI

Two modes: Player vs Player or Player vs AI. The AI uses Minimax with Alpha-Beta Pruning and is completely unbeatable. Watch node evaluations and tree depth update in real time. Includes undo, score tracking, and hover move preview.

# Backtracking

Sudoku Solver

Fully editable 9×9 grid with 3 preset difficulty levels. The backtracking solver animates every placement and backtrack step. Blue cells = current attempt, red = backtracking. Adjustable animation speed.

⊞ A* Search

Maze Pathfinder

Draw walls on a canvas grid, place start/end points, and watch A* explore the maze. Color-coded open set and visited nodes. Random maze generator using DFS recursive carving. Multiple grid sizes.

♛ Backtracking

N-Queens Solver

Configurable from 4 to 12 queens. Animated backtracking shows queen placements, conflict detection (red queens), and removals. "All Solutions" button finds every valid configuration. Navigate between solutions with prev/next.
