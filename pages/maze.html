⭐ A* Search Algorithm
▾

A* (A-star) finds the shortest path from start to goal. It combines Dijkstra's algorithm with a heuristic to guide the search efficiently.

f(n) = g(n) + h(n) g(n) = cost from start to n h(n) = heuristic: Manhattan dist f(n) = estimated total cost
📐 Manhattan Heuristic
▾

Manhattan distance counts horizontal + vertical steps between two grid points:

h(n) = |n.x - goal.x| + |n.y - goal.y|

It's admissible (never overestimates) so A* is guaranteed to find the optimal path.

📋 Algorithm Steps
▾
  1. Add start node to the open set (priority queue)
  2. Pop node with lowest f(n) from open set
  3. If it's the goal: reconstruct and return path
  4. Add to closed set (already explored)
  5. For each valid neighbor: compute g and h scores
  6. If neighbor not in closed set or has better g: update and add to open set
  7. Repeat until goal found or open set empty

▾
Time O(E log V)
