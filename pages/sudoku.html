↩️ Backtracking Search
▾

Backtracking is a systematic search that incrementally builds candidates and abandons them ("backtracks") as soon as a constraint is violated.

For Sudoku: try placing 1–9 in each empty cell. If placing a number violates any constraint (row/col/box), immediately try the next number.

📋 Algorithm Steps
▾
  1. Find the next empty cell (index scan)
  2. Try digits 1 through 9 in that cell
  3. Check: is digit valid in this row, column, and 3×3 box?
  4. If valid: place digit and recurse to next empty cell
  5. If recursion returns false: erase digit (backtrack)
  6. If all digits fail: return false (propagate backtrack)
  7. If no empty cells remain: puzzle is solved!
⏱ Complexity
▾
Worst case O(9^81)
Typical (pruned) Much less
Space O(81)
Constraints checked Row · Col · Box
